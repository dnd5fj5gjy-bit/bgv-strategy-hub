Bear Witness Dashboard

Ultimate Social Media Management Dashboard — dark premium React app with 8 interconnected modules, AI strategy engine, and cross-portfolio client management.

This is a live application, not a static document. It runs as a React frontend + Node backend on the Mac Studio. Access from outside the studio's local network requires Tailscale (or a public tunnel if exposed).

What it does

  • Cross-portfolio social management across BGV brands (Modern Savage, Ted's Health, Sister Directory, BMF)
  • AI strategy engine producing content briefs and calendar plans
  • 8 interconnected modules: clients, calendar, content, briefs, analytics, narratives, library, settings
  • Dark, premium UI — designed to read as a serious operator tool, not a content-mill

Access

Local (on Mac Studio LAN):
Typically served at http://localhost:<port> with the backend running.
Remote (Olly):
Tailscale is required — see Nexus / Felix for set-up. Direct internet access has not been provisioned for this dashboard.
