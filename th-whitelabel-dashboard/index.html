Ted's Health — White-Label Partner Command Centre

Pipeline CRM with 61 pre-loaded prospects, AI outreach drafting, full partnership agreement template, LegitScript shield feature, and analytics dashboard.

This is a live application, not a static document. It runs as a React frontend + Node backend on the Mac Studio with a SQLite store. Access from outside the studio's local network requires Tailscale.

What it does

  • 61-prospect pipeline with LegitScript certification status (12 certified, 39 not-certified, 10 unknown)
  • AI outreach drafts per prospect using TH WL context briefs
  • Full white-label partnership agreement template, ready to send
  • Analytics: pipeline stages, contact recency, partner conversion rate
  • LegitScript shield badge feature on each prospect card

Access

Local (on Mac Studio LAN):
Typically served at http://localhost:<port> with the backend running.
Remote (Olly):
Tailscale is required — see Nexus / Felix for set-up. Direct internet access has not been provisioned for this dashboard.
